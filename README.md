# Student Management System

A simple console-based Student Management System built with Python.

## Features
- Add student information
- Display student details
- Update CGPA
- Add phone number
- Delete semester
- Menu-driven program

## Concepts Used
- Dictionary
- While loop
- If/elif/else
- Break
- Continue
- User input
